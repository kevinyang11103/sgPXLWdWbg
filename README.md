# 图书信息管理系统 python33

本项目为一套基于Python语言及其图形用户界面库PyQt5和轻量级数据库SQLite3开发的图书信息管理系统。以下是系统的详细概述：

## 技术栈

- **后端**: Python 3
- **前端**: PyQt5
- **数据库**: SQLite3

## 功能模块

### 基本功能

- 显示图书表中所有图书的相关信息
- 将图书信息插入到指定位置
- 将指定位置图书删除

### 高级功能

- 图书去重
- 图书信息的模糊查询
- 图书的批量修改

### 用户系统

- 登陆注册系统
- 权限管理系统

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- 操作系统：兼容Windows、Linux和macOS
- Python版本：3.x

## 部署步骤

1. 确保已安装Python 3.x环境
2. 安装PyQt5库和SQLite3
3. 解压源码和UI文件
4. 运行主程序
